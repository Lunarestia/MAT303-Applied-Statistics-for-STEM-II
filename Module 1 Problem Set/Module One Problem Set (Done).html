Step 6: Multiple Regression¶

In this step, you will create a multiple regression model for fuel economy as the response variable, and weight and quarter mile time as predictor variables. You will print summary statistics of this model that will help evaluate the significance and utility of the model.

The general form of this regression model is:



\begin{equation*} \large E(Y) = {\beta}_0\ +\ {\beta}_1\ {x}_1\ +\ {\beta}_2\ {x}_2 \end{equation*}

\begin{equation*} \text{where y is fuel economy,}\ {x}_1\ \text{is weight, and}\ {x}_2\ \text{is quarter mile time} \end{equation*}

Click the block of code below and hit the Run button above.

In [8]:
# Create the multiple regression model and print the statistics
model <- lm(mpg ~ wt + qsec, data=mtcars_subset)
summary(model)

Interpretation of Beta Estimates¶

Suppose the regression model is:

\begin{equation*} \large E(y) = 19.7462\ -\ 5.0480\ {x_1} +\ 0.9292\ {x_2} \end{equation*}

Interpret the estimated coefficient of weight variable.

  • The estimated coefficient for variable weight is -5.0480. This means that on average, the fuel economy of the car drops by 5.0480 for each unit increase in the weight. Note that the variable weight is expressed in 1,000lbs (for example, a weight of 3.45 represents 3,450lbs). Therefore, the fuel economy of the car drops by 5.0480 for each 1,000lbs increase in the weight.
